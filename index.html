свойства флекс

display:flex
Назначение: включает функию flex.
Возможные значения: flex, inline flex
flex-direction: row
Назначение: расположение по осям x y
Возможные значения: row, row-reverse, column, column-reverse
flex-wrap: nowrap
Назначение: делит элементы по строкам

Возможные значения: wrap, nowrap, wrap-reverse

flex-flow: row nowrap
Назначение: принимает функции flex-direction flex-wrap
justify-content: flex-start
Назначение: перемещение по горизонтали

Возможные значения: flex-star, flex-end, centre, space-between, space-around

align-items:
Назначение: перемещение по vertical

Возможные значения: flex-star, flex-end, centre, stretch, baseline
